ECHO :
- les caractères spéciaux ne sont pas correctement interprétés
- finir l'option -n
- retester : parfois un même argument est affiché plusieurs fois

GENERAL :
- les " doivent toujours être par deux
- les ' peuvent être seules si dans "
- le parsing ; ne doit pas se faire si compris entre " 
	ex : echo "ca va ; echo 1"		!=		echo "ca va" ; echo 1
